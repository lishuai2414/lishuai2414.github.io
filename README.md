# lishuai2414.github.io

基于 React + React-router + ES6 前端项目


## 目录结构

```
 -assets/             # 静态文件
     |- index.chunk.js 个人主页内容
```

## 使用方法

```sh
$ git clone https://github.com/lishuai2414/lishuai2414.github.io.git
$ cd lishuai2414.github.io
$ npm install
$ npm run dll & npm start
```
